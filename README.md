## E-commerce Copywriting Skill

Write copy that converts browsers into buyers. Covers every touchpoint in an online store:

- **Product descriptions** — Templates for fashion, electronics, food, B2B
- **Bulk descriptions** — Modular system for 50+ products
- **Checkout flow** — Cart → shipping → payment → confirmation
- **Transactional emails** — Order confirmation, shipping, reviews, abandoned cart
- **Trust signals** — Shipping, returns, security badges, social proof
- **E-commerce SEO** — Title tags, meta descriptions, schema markup
- **Category pages** — Navigation and SEO copy
- **Empty states** — Cart, 404, out of stock

### What it adds to the original copywriting skill

The [original copywriting skill](https://github.com/coreyhaines31/marketingskills) focuses on marketing landing pages. This extends it with e-commerce-specific patterns for Shopify, WooCommerce, Next.js storefronts, or custom platforms.
